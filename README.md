Практическое задание номер 1 по курсу "Мультиагентные технологии". Выполнил Вековцев Вячеслав Владиславович.

Принцип работы алгоритма по этапам:
Этап 1: каждый агент имеет данные о своём номере, своём числе и о связи с другими агентами.
Этап 2: каждый агнет предеаёт (и соответственно получает) каждому доступному агенту свой номер и своё число.
Этап 3: этап 2 прододжается до тех пор, пока все агенты не узнают обо всех. После достижения консенсуса алгоритм останавливается.
Этап 4: каждый агент вычисляет среднее арифметическое.

Алгоритм запуска программы (для среды разработки IntelliJ IDEA Community Edition):
1. Скопирывать ссылку на гитхаб из письма.
2. В гитхабе скачать файлы jade.jar и main.java.
3. Открыть среду разработки IntelliJ IDEA. Выбрать File, далее New..., далее Project from version control. Вставить ссылку на гитхаб и нажать clone.
4. Открыть File, Project Structure, Modules, Dependencies, нажать на плюс и добавить JARs or directories. В открывшемся окне указать путь к jade.jar. Нажать ОК.
5. В файле task1.jpg можно ознакомиться с топологией заданного графа.
6. Можно запускать код.
